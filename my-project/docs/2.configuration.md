#2. Configurations

1. **Installer l'Application :**
   - Téléchargez l'executable de l'application
        - .app pour les appareils Mac
        - .exe pour les appareils windows
2. **Lancer l'Application :**
   - Ouvrez Safari Go! sur votre appareil.
